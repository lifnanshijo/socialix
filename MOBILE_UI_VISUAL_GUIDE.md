## Responsive Breakpoints

### < 600px (Mobile)
- Full-width layouts
- Bottom navigation visible
- Hamburger menu active
- 12px padding
- 1-column content
- Smaller images
- Touch-friendly sizes

### 601-900px (Tablet)
- Hybrid layout
- Responsive spacing
- Medium padding
- Flexible columns
- Scaling font sizes

### > 900px (Desktop)
- Original layouts
- Max-width containers
- Full spacing
- Multi-column
- Original font sizes

## Interaction States

### Button States
```
Normal:    Primary color, no shadow
Hover:     Slight background change
Active:    Darker shade
Disabled:  60% opacity
Focus:     Border highlight
```

### Navigation States
```
Current:   Blue highlight + icon
Inactive:  Gray icon
Hover:     Light background
Active:    Blue icon + label
```
